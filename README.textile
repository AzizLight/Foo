h2. Foo

Foo is intended as a replacement for print_r and var_dump. It provides a richer
formatted output that can be output directly to the screen, or to a javascript
console.

h2. Usage

Foo is comprised of 3 functions:

| Function | Display | Notes
|   foo    | Screen  | Output any type (string / array / object, etc.). Includes
					   a full debug backtrace
|   foc    | Console | Output any type, no backtracing.
|   fo     | Screen  | Output a single line, only works with strings / numbers, no backtracing
